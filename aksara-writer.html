Aksara Writer

Modern markdown-to-document converter with Indonesian business templates. Export to PDF, PPTX, and HTML with professional styling.

Quick Start

Get started with Aksara Writer in seconds:

npm install -g aksara-writer

Current Version: v0.1.2 with 4 themes, VSCode auto-preview, and improved styling

Key Features

Multiple Export Formats

Export your markdown documents to PDF, PowerPoint (PPTX), and HTML with consistent professional styling.

Professional Themes

Choose from 4 carefully designed themes optimized for Indonesian business documents and presentations.

VS Code Integration

Seamless integration with Visual Studio Code featuring live preview, auto-export, and template insertion.

Indonesian Language Support

Built-in support for Bahasa Indonesia with proper typography, formatting, and cultural context.

Bun Runtime

Powered by Bun for lightning-fast document processing and conversion speeds.

Flexible Configuration

Customize output with Aksara directives for document metadata, styling, and layout options.

How It Works

1

Write Markdown

Create your content using standard markdown syntax with Aksara directives for enhanced formatting.

2

Choose Format

Select your desired output format: PDF for documents, PPTX for presentations, or HTML for web.

3

Export & Share

Generate professional documents ready for business use, complete with Indonesian formatting standards.

# Basic Usage Example aksara-writer convert document.md --format pdf --theme business aksara-writer convert presentation.md --format pptx --theme modern

Architecture

Built as a modern TypeScript monorepo with three interconnected packages:

Core Engine

packages/core - TypeScript conversion engine with Puppeteer PDF generation and JSZip PPTX creation

CLI Tool

packages/cli - Command-line interface built with Commander.js for batch processing and automation

VS Code Extension

packages/vscode - Editor integration with live preview, template gallery, and one-click export
