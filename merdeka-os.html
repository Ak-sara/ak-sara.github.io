Project Overview

MerdekaOS is a purpose-built Linux distribution designed specifically for Indonesian users. Based on Arch Linux, it provides a lightweight, efficient operating system pre-configured with Indonesian language support and bundled with the complete Ak'sara software ecosystem.

Development Timeline

Target: 12-18 months to stable release
Current Status: Architecture planning and base system design
Phase: Long-term vision project following Aksara IS completion

Key Features

Lightweight Design

Optimized for performance with minimal resource usage, perfect for older hardware and efficient workflows.

Indonesian-First

Complete Bahasa Indonesia localization, Indonesian keyboard layouts, and culturally appropriate default settings.

Pre-bundled Software

Ships with Aksara Writer, development tools, and essential applications ready for Indonesian business use.

Office-Optimized

Pre-configured for typical office workflows with document processing, communication tools, and business applications.

Developer-Friendly

Complete development environment with popular programming languages, editors, and development tools pre-installed.

Secure by Default

Enhanced security settings, firewall configuration, and privacy-focused defaults for business and personal use.
